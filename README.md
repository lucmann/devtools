devtools
========

# Purpose

This is a helper script which assists in installing, uninstalling and configuring
the productive tools listed below.

-	ack
-	ag
-	autojump
-	cmake
-	cscope
-	ctags
-	fzf
-	gcc
-	g++
-	gdb
-	git
-	meson
-	ohmyzsh
-	pip3
-	tmux
-	tpm
-	vimrc
-	zsh

NOTE: These above are tested on WSL2 Ubuntu 20.04
